# 🐸 starkle's filter lists

Opinioned web content filters by me. If they seem useful, you are welcome to try them.

## Usage

Copy the raw github URL for your desired filter list(s) and import it into your content filter.

The following combinations are tested by me:

- Firefox + uBlock Origin
- Brave
- Brave for Android

## Philosophy

I wanted filter lists that hide distracting and addictive content, to avoid the need for browser extensions that claim to do the same thing.

The more annoyances I blocked, the more I noticed a pattern. Almost all could be categorized as one of the following anti-features:

- Feeds
- Suggestions
- Elements requiring an account
- Elements requiring payment

I try to keep my rules in these categories, rather than just my personal "annoyances".

Eradicating these still allows any one piece of content to be accessible and searchable, but significantly reduces the risk of attention traps and habit formation. Plus it just looks cleaner.
